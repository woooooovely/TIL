# CRUD?

CRUD는 대부분 컴퓨터 소프트웨어가 가지는 기본적인 데이터 처리 기능인 **Create(생성), Read(읽기), Update(업데이트), Delete(삭제)**를 묶어서 일컫는 말이다. UI가 갖추어야 할 기능(정보의 참조/검색/갱신)을 가리키는 용어로서도 사용된다.
